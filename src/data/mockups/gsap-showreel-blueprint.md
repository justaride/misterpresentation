# Presentation Blueprint: GSAP Showreel

## ID
`gsap-showreel`

## Overview
High-impact animation showreel powered by GSAP ScrollTrigger and timeline sequences.

## Metadata
- Category: `animation-heavy`
- Format: `scrollytelling`
- Difficulty: `advanced`
- Tags: `gsap`

## Visual Style
- Typography: `var(--font-display)` for big headings, `var(--font-mono)` for UI labels, `var(--font-body)` for supporting text.
- Palette: bold gradients and high-saturation accents to make motion legible.
- Signature elements:
  - GSAP: ScrollTrigger timelines and scrubbed motion.

## Navigation & Controls
- Scroll: native scrolling drives progress.
- Replay: scroll back up to re-run reveals.
